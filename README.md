# PARI — Protected Areas of India

An interactive map explorer for India's national parks, wildlife sanctuaries, tiger reserves, and conservation areas. Built for visitors planning trips, researchers looking for datasets, and anyone curious about what conservation initiatives are happening on the ground.

Browse park boundaries on a searchable map, filter by state or designation, and explore each area's visitor info, research resources, and active conservation programmes — all in one place.

**Stack:** TypeScript, React, MapLibre GL JS, Fastify, PostgreSQL + PostGIS  
**Data:** WDPA (World Database on Protected Areas) via Protected Planet
